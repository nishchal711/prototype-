# College Attendance App Prototype

## Project Overview
This project is a prototype model of a College Attendance Management Application designed using Figma.

## Features
- Student Login
- Teacher Dashboard
- Attendance Marking
- Attendance Reports
- Profile Management

## Tools Used
- Figma
- GitHub
